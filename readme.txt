This widget randomly displays a snapshot of a website from your blogroll.


== Installation ==

1. Unzip the archive.
1. Upload all files to the `/wp-content/plugins/snap-my-roll/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Make sure that your upload directory `/wp-content/uploads` is writable
1. Add the Snap My Roll widget to your sidebar
1. Edit parameters according to your needs
